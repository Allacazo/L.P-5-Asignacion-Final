Tarea #5 Asignacion Final:

Explicacion: Este programa ha sido creado en c++, Este permite registrar las claificaciones de maximo 20 estudiantes y 4 diferentes calificaciones. Luego muestra un resumen agregando el promedio sumado del estudiantes en solitario, Ademas se podra ver un resumen de Promedio general del Grupo, La calificacion mas alta y mas baja, y la cantidad de estudiantes que aprovaron y desaprobaron.

El programa funciona primero pidiendo que ingrese la cantidad de estudiantes que quieres registrar (El programa mismo aclara que es un maximo de 20) Esto es para saber la cantidad de veces que se repetira el proceso, Pasado esa parte el programa solicitara ingresar el nombre del estudiantes deseado para luego proseguir con el ingreso de las cuatro calificaciones (El programa aclarara que solo acepta digitos del 0 a 100), Este proceso se debe de repetir la cantidad de veces que se especifico al ingresar la cantidad de estudiantes. Despues de completar cada uno de estos procesos el programa le hara un resumen por orden de ingreso donde se mostrara los estudiantes, Sus 4 calificaciones y su promedio.
Pero mas al final de todo se mostrara una pequeña parte que monstrara datos unicos los cuales son: Promedio general del Grupo, La calificacio mas alta y mas baja, y la cantidad de estudiantes que aprovaron y desaprobaron.

Como funcionan los arreglos en este programa: 
El Programa utiliza arreglos unidimensionales que guardan la informacion de cada estudiantes. 
1. El arreglo Nombres[] se usa para almacenar el nombre de cada estudiantes.
2. El arreglo Promedio_Por_Estudiantes[] Guarda el promedio final de cada uno.
3. Y por ultimo el arreglo Notas[] se usa para guardar todas las calificaciones del grupo en un solo arreglo.

Tambien como cada estudiantes tiene 4 notas, Se usa una formula para colocar cada calificacion en su posicion correcta del arreglo expecificamente.
